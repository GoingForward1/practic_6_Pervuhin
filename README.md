Первухин Никита ET-112.
Вариант 20.
Задание на массивы:
Дана матрица точек размером NxM. Известны размеры квадратной матрицы 
KxK (K<N, K<M). Вывести на экран все квадратные матрицы, полученные из 
исходной матрицы.

Тесты:
Входные данные: n, m - размерность исходной матрицы. k(<n;<m) - порядок квадратной матрицы.
Выходные данные: квадратные подматрицы k-го порядка находящиеся в исходной матрице nxm.
